## 新サービス追加手順

1. このファイルに行を追加（ステータス: 🔴 準備中）
2. `autoBlog/blog-[name]/` ディレクトリ作成・git init
3. カラフルボックスでドメイン取得 → WP インストール
4. Search Console 登録（手動）
5. `autoBlog/_factory/daily_report.py` の SITE_LIST に追加
6. ステータス → 🟢 運用中

## ステータス凡例

- 🔴 準備中（ドメイン取得前）
- 🟡 設計中（WP設定・初期記事生成前）
- 🟢 運用中（自動投稿・SEO監視稼働）
- ⛔ 停止中
